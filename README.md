# S3 Discord Embedder
A poorly written Flask app that makes fancy Discord embeds from S3 objects, aye!

### `config.json`
```json
{
    "app_url": "https://cdn.example.com",
    "s3_endpoint": "https://example.fra1.digitaloceanspaces.com",
    "bind_address": "0.0.0.0",
    "bind_port": 80
}
```
